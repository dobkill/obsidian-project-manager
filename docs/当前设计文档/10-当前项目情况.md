# 10. 当前项目情况

## 1. 当前状态

当前项目已经从最初的基础 MVP，升级为一版围绕总览体验、重复任务模型和组合任务表达能力重构后的可编译插件。

## 2. 本轮完成的重点

1. 重复任务存储改为“单条系列记录 + 发生日期集合”。
2. 项目进度页中重复任务改为单行展示，不再拆成多条实例。
3. 周任务图每日卡片增加最小高度，空周不再塌陷。
4. 任务总览页新增最近 30 天任务趋势折线图。
5. 今日任务页和周任务图改为按运行时展开的实例处理勾选、删除和结束重复。
6. 热度图与趋势图统一按真实完成时间统计。
7. 初始化阶段增加数据读取失败保护，避免损坏文件被空数据自动覆盖。
8. 修改数据目录时会优先加载目标目录已有的有效插件数据；目标目录不存在、为空或插件数据格式错误时，会用当前内存数据创建新文件。
9. 项目进度页编辑重复任务时移除“删除本次”，只在实例视角提供单次删除。
10. 任务弹窗按重复类型条件显示字段，并移除系列级“完成状态”开关。
11. 任务弹窗新增“组合任务”类型，并支持在弹窗内增删子任务。
12. 周任务图和今日任务页支持“一个大框 + 多个子任务小框”的组合任务展示与完成交互。
13. 组合任务完成度按子任务比例计算，并同步体现在项目进度页和今日进度条中。
14. 趋势图纵轴改为自动计算的整数刻度，图例文字颜色与折线颜色一致。
15. 数据变更改为先完成 JSON 写入再刷新视图，插件卸载时会等待待写入队列并补写当前快照。
16. 自定义数据目录的读写链路补强：插件启动时会主动从当前数据目录加载，workspace ready 后再补一次回读，目录切换前会先冲刷旧目录待写入队列，数据修改后会立即回读磁盘内容，避免“已存数据但面板未显示”。
17. 已核查“无所属项目任务”存储链路，确认今日任务页和周任务图中新建的未归属项目任务会正常写入本地。
18. 点击“项目总览”和“今日任务”入口时，会先显示缓存再异步回读磁盘，保证启动后首次打开也能尽快与文件数据对齐。
19. 数据目录创建改为幂等处理，修复启动阶段偶发的 `folder already exists` 异常。
20. 周任务图任务卡片改为紧凑模式，重复徽标弱化为 chip，编辑入口改为轻量图标并减少卡片阴影和边框厚度。
21. 任务模型新增状态、优先级、标签、单次发生覆盖、四视图状态、来源链接、任务笔记和 revision。
22. 项目管理页新增表格、看板、甘特图、思维导图四视图。
23. 支持格式化文本批量导入任务，并提供导入预览。
24. 支持右侧快速记录页，可写日记、追加任务笔记、快速创建任务、扩充思维导图。
25. 支持全局 Markdown 项目任务块后台扫描，并写入 `note-task-index.json`。
26. 支持今日自动排程，排程记录写入 `write-history.json`。
27. 今日任务页重构为聚焦式卡片布局，新增环形进度卡、优雅空状态和菜单化危险操作。
28. 快速记录页重构为 Quick Capture 面板，支持工具栏、文件/任务选择器，以及“追加正文 / 创建子节点”两种导图写入模式。
29. 项目进度页新增整体进度摘要卡，统一 Button / Tabs / Badge 视觉规范。
30. 项目表格视图改为卡片化表格，补齐 badge、完成度条、分页和操作菜单。
31. 项目看板视图支持四列拖拽、底部统计条和阻塞依赖提示。
32. 项目甘特图改为图形化日期时间轴，并支持里程碑与锁定状态切换。
33. 项目思维导图补齐真实可用的节点编辑链路，支持任务/评语关系调整与评语改写。
34. 快速记录页的“补充导图”改为先选项目、再选项目内节点，并新增跨任务共享的快捷节点区。
35. 快速记录四个目标卡片改为稳定的 grid 布局，修复侧边栏缩放时文案与卡片错位问题。
36. 项目思维导图修复节点拖拽后强制回到全局视角的问题，保留当前缩放与平移状态。
37. 项目思维导图任务节点新增正文摘要展示，追加到节点正文后可直接在导图中看到结果。
38. 批量导入弹窗与设置页示例补齐“已完成任务”导入说明，并在预览区直接展示已完成数量。
39. 新建任务与格式化文本导入在遇到时间冲突时，改为自动寻找同日 1 分钟空档占位，不再直接报冲突错误。
40. 逾期未完成的单次任务会在数据加载后自动顺延到今天，并自动占用不冲突的 1 分钟时间槽。
41. 今日任务页的“导出今日任务”改为直接复制“今日未完成任务”，移除三档导出菜单。
42. 项目甘特图重构为专业时间轴布局，新增左表格对齐、日/周/月粒度、缩放、适应、today 线与 mini map。

## 3. 数据结构变化

当前版本不兼容旧任务结构。

新任务模型核心字段变为：

1. `occurrenceDates`
2. `occurrenceStates`
3. `kind`
4. `subtasks`
5. `date` 作为系列起始日期
6. `recurrenceCount`
7. `recurrenceUntil`
8. `status`
9. `priority`
10. `tags`
11. `occurrenceOverrides`
12. `viewState`
13. `sourceLinks`
14. `notes`
15. `revision`

## 4. 已验证情况

当前代码已通过本地构建：

```text
npm run build
```

并通过 TypeScript 检查：

```text
node node_modules/typescript/lib/tsc.js --noEmit
```
