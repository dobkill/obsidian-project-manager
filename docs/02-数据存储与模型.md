# 02. 数据存储与模型

## 1. 存储目录

插件数据默认存放在：

```text
project-manager-data/
```

目录结构：

```text
project-manager-data/
├── config.json
├── projects.json
├── progress-pages.json
└── tasks/
    ├── 2026-05.json
    ├── 2026-06.json
    └── ...
```

## 2. 配置模型

`config.json` 关键字段：

```ts
type PluginConfig = {
  version: string;
  dataFolder: string;
  overviewTab1Name: string;
  overviewTab2Name: string;
  weekStartsOn: "monday";
  timeSlotMinutes: number;
  heatmapRange: "12months";
  showCompletedTasks: boolean;
  defaultTaskDurationMinutes: number;
  defaultTaskStartTime: string;
};
```

其中：

1. `defaultTaskStartTime` 用于当天没有任何排期任务时的默认开始时间。
2. `defaultTaskDurationMinutes` 用于自动推算结束时间。

## 4. 任务模型

当前任务模型已经升级，不兼容旧版多实例重复任务结构。

```ts
type Task = {
  id: string;
  title: string;
  description?: string;
  projectId?: string;
  date: string;
  startTime?: string;
  endTime?: string;
  recurrence: "once" | "daily" | "weekly";
  recurrenceCount?: number | null;
  recurrenceUntil?: string | null;
  occurrenceDates: string[];
  completedOccurrences: Array<{
    date: string;
    completedAt: string;
  }>;
  createdAt: string;
  updatedAt: string;
};
```

### 4.1 字段说明

1. `date` 表示这条任务系列的起始日期。
2. `occurrenceDates` 记录全部发生日期。
3. `completedOccurrences` 记录已完成的发生日期，以及对应完成时间。
4. `recurrenceCount` 和 `recurrenceUntil` 仍然用于描述重复边界。

### 4.2 设计含义

1. 单次任务也是一条系列任务，只是 `occurrenceDates` 只有一天。
2. 重复任务在存储层不再拆成多条实例。
3. 页面展示需要的实例数据，在运行时根据 `occurrenceDates` 展开。

## 5. 运行时实例模型

页面层会把 Task 展开为临时 TaskOccurrence：

```ts
type TaskOccurrence = {
  id: string;
  taskId: string;
  occurrenceDate: string;
  occurrenceNumber: number;
  date: string;
  completed: boolean;
  completedAt?: string | null;
  // 其余标题、时间、项目字段继承自 Task
};
```

用途：

1. 今日任务页按 `date === 今天` 读取。
2. 周任务图按周内日期读取。
3. 热度图与趋势图按 `completedAt` 的日期统计。

## 6. 项目进度页模型

```ts
type ProgressPage = {
  id: string;
  projectId: string;
  name: string;
  columnOrder: string[];
  createdAt: string;
  updatedAt: string;
};
```

当前默认列意图为：

```text
任务名称 -> 重复 -> 计划 -> 完成 -> 描述 -> 操作
```

## 7. 月度任务文件

每个月一个文件：

```ts
type TasksFile = {
  month: string;
  tasks: Task[];
};
```

`month` 由任务起始日期所在月份决定。

## 8. 当前选择

当前实现没有兼容旧格式迁移逻辑，原因如下：

1. 重复任务存储结构已经彻底变化。
2. 项目进度页的核心诉求就是“重复任务单行展示”。
3. 当前项目仍处于可大改阶段，直接切新结构更清晰。
