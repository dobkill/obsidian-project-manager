# 08. 设置项与版本规划

## 1. 当前设置项

当前设置页支持以下内容：

1. 数据目录路径。
2. 任务总览标签名称。
3. 项目进度标签名称。
4. 时间粒度。
5. 默认任务时长。
6. 默认开始时间。
7. 是否显示已完成任务。

## 2. 当前版本完成的重构

当前版本已经完成以下方向：

1. 任务总览页主内容区打开。
2. 热度图日期标识增强。
3. 周任务图列头重做。
4. 新增任务默认时间智能推算。
5. 项目进度页“完成”列前置。
6. 重复任务支持。
7. 系列级删除与完成支持。

## 3. 后续建议

### 3.1 下一阶段

1. 更完整的热度图弹窗详情。
2. 项目进度页筛选。
3. 项目进度页排序。
4. 周任务图拖拽排期。

### 3.2 中期阶段

1. 跨天任务。
2. 自定义字段。
3. 标签与优先级。
4. 任务依赖关系。

### 3.3 后期阶段

1. 数据备份与恢复。
2. 导入导出。
3. 与其他 Obsidian 任务生态集成。
4. 移动端专项优化。

## 4. 当前明确放弃的目标

在当前代码基础上，已不再追求旧格式兼容。

这意味着：

1. 旧任务 JSON 结构不是设计约束。
2. 后续文档以新模型为准。
