# 09. 设计盲区与未解决问题

本轮大改已经解决了“总览不聚焦”和“重复任务缺位”两个核心问题，但仍有一些未完成项。

## 1. 周任务图仍然不是时间网格

当前周任务图已经完成：

1. 日期列重做。
2. 今天高亮。
3. 过去日期灰显。
4. 默认时间智能推算。

但它仍然不是拖拽式时间网格日历。

## 2. 重复任务仍是预生成模式

当前版本采用“创建时生成未来实例”的方案。

优点：

1. 逻辑直观。
2. 今日未完成不会覆盖历史。
3. 系列级操作好理解。

缺点：

1. 长周期重复任务会提前生成较多数据。
2. “修改整个系列从第一天开始”当前没有独立入口。

## 3. 热度图详情仍然偏轻

热度图点击后已经从通知升级为弹窗，但仍然是基础列表，不是完整详情页。

## 4. 项目进度页仍缺少筛选排序

当前项目进度页重点放在：

1. 完成列前置。
2. 重复任务管理。
3. 统一编辑规则。

但还缺少：

1. 筛选。
2. 排序。
3. 批量操作。

## 5. 数据迁移未实现

当前版本明确不兼容旧版 JSON 结构，因此：

1. 没有旧数据迁移器。
2. 也没有双格式读写逻辑。
